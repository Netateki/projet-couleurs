# Projet jeu de couleurs

Jeu de couleurs interactif, dont le but est d'appuyer sur la couleur du mot et non la couleur que le mot décrit.

## Utilisation

Executez le code dans un interpréteur ou dans un terminal/invité de commandes.
Le bouton démarrer sert à lancer le décompte et à commancer la partie.
Le bouton réinitialiser sert à réinitialiser le jeu ainsi que le décompte.
